## 0. 최상위 공통 법칙 (항상 로드 · 전 레포 공통)

- a) **백그라운드 실행 = 착수·계획 단계 사전 승인.** 디폴트 = 포그라운드. 턴이 끝나도 계속 도는 실행(`npm run dev` 서버 상주 · Bash `run_in_background` · 백그라운드 에이전트·와처)은 운영자 승인 후에만 — 필요 여부가 정해지는 착수·계획 단계에 AskUserQuestion 버튼으로 (무엇을 · 왜 · 언제 끝나는지) 묶어 **한 번에** 묻는다(실행 직전 낱개 질문 반복 금지 · 운영자가 명시로 시킨 백그라운드 = 그 지시가 곧 승인 · 중간에 새로 필요해지면 실행 전에 묻기 · 완료·중단 시 남은 프로세스 정리). **강제(기계·A안)** = PreToolUse 훅 `.claude/hooks/bg_gate.py`(`.claude/settings.json` 등재)가 백그라운드 Bash를 무조건 승인 팝업으로 강제(백그라운드 아님 = 무의견·오차단 0).
- b) **완료 시점 예고 금지.** *1~2분 뒤 완료된다/반영된다* 류 소요·완료 시간 예측을 응답에 쓰지 않는다 — 완료는 실측 확인 후에만 보고(전파 대기가 낀 변경은 시간 약속 대신 확인 방법을 준다).
- c) **머지 = 타 세션 충돌 점검 후 · 머지 뒤 운영자 확인 요청.** 머지 전 최신 origin/main 재동기화로 다른 세션 작업과 충돌·역행 없는지 확인하고, 머지·read-back 후 보고에 운영자가 직접 확인할 포인트 1줄과 확인 요청을 명시.

## 1. 레포 정체 (실측)

- a) 페이지 정본 = `src/pages/index.astro` 단일 파일(인라인 CSS/JS). `src/drafts/` = 빌드 비대상 보관(라우팅 안 됨).
- b) **디자인 토큰 = `src/pages/index.astro`의 `:root`** — nomute-editor viewer `:root` 이식(값 복사 스냅샷 · 개념 정본은 에디터). 색·radius·간격·blur·타이포·모션·눌림 스케일 전부 여기. 디자인 값은 이 토큰 **계승이 디폴트, raw 즉석 창작 금지** — 홈 전용 축이 필요하면 홈 토큰으로 신설(`--fs-hero` 선례).
- c) RSS 편성 = `src/lib/feeds.ts`(소스별 큐→중복제거→성향 슬롯) + `src/lib/rss.ts`.
- d) 빌드 = `npm run build`(astro build) · 개발 = `npm run dev`. 배포 = Cloudflare Pages(프로젝트 `nomute` — 커밋·PR마다 프리뷰 URL 자동 코멘트 · 빌드 실패는 PR에서 바로 보임). 전용 디자인 게이트 스크립트 없음 — 빌드 통과 = 최소 게이트, 토큰 규율은 사람이 지킨다.
- e) 반응형 = 모바일 우선 + 데스크톱 브레이크 `@media (min-width: 768px)` 하나(히어로 타이포 `--fs-hero` 46px / `--fs-hero-pc` 72px 2티어).

## 2. 🎛 디자인 제안 = 플레이그라운드식 HTML (항상)

디자인·UI 제안/시안/튜닝 = **플레이그라운드식 HTML이 디폴트**(정적 이미지·텍스트 나열로 갈음 금지). 산출 = `docs/reports/{yymmdd}_{라벨}.html` 자기완결 1파일(자산 base64 임베드 · 외부 요청 0 · 인라인 CSS/JS).
구성 5요소(전부 필수):
- ① 실물 재현 미리보기 — 실제 화면 프레임(폰 세로 프레임 기본 + 데스크톱 ≥768px 와이드 토글 — 반응형 홈이라 양쪽이 실사용) 안에 `src/pages/index.astro`의 `:root` 토큰 사본 + 정본 컴포넌트·모션 계승으로 재현. **현행 상태 = 비교 기준으로 함께 재현(고정)** · 튜닝 변수 기본값 = 현행 실측값.
- ② 프리셋 3~4개 — 현행 포함 · 추천 ★ 1개.
- ③ 미세조정 컨트롤 — 축별 그룹 라벨 · 값 표시 = tabular-nums · **옵션 규모 = 평균 50개 안팎**(항목 최대 고도화 — 단순 대상만 축소 허용) · **축 3분류(기틀 인지)**: {정렬·간격·높이 = 자유} · {글자·라운드 등 확립 축 = 자유 조절 + 실존값 사다리 눈금, 사다리 밖 = ⚠ 실시간} · {색 = 팔레트/토큰 폐쇄 셀렉트(자유 hex 금지) — 역할 재지정 = ⚠} · **복사 시 ⚠ 축 존재 = 알림창 + 「기틀 갱신 역제안」 블록 포함**(회신 = 기틀 갱신 승인 정문 경로 · 같은 토큰 알파 변주 = 자유).
- ④ 선택값 출력 + 복사 버튼 — 값마다 현행 병기 · 기존 토큰 동값 = "계승" / 아니면 "갱신 후보" 자동 표기 · 기틀 축 = ⚠ 딱지.
- ⑤ 재현 한계 각주 — 폴백·근사·실물과 다른 점 명시.
검증 = 헤드리스 브라우저 렌더 실측(상태 전환 실클릭 + 프리셋별 스크린샷 + JS 에러 0) 후에만 전달. 전달 = 레포 커밋(정본) + **채팅 파일 직접 첨부**(원탭) · Artifact 링크 = 같은 시안 반복 수정 루프·외부 공유 시 보조.
후속 = 운영자가 [선택값 복사] 결과를 붙이면 그 값을 `src/pages/index.astro`의 `:root`·CSS에 배선(+`npm run build` 통과)까지.
